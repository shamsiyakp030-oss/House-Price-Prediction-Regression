# House Price Prediction using Machine Learning

##  Overview
This project focuses on predicting house prices using regression algorithms based on features like bedrooms, bathrooms, and area.

---

##  Goal
To predict house prices using regression models based on housing features.

---

##  Dataset
- Source: Kaggle  
- Dataset: House Sales Prediction  
- Features include:
  - Bedrooms
  - Bathrooms
  - Square Foot Area
  - Location
  - Price

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

##  Steps Involved

### 1. Data Collection
Dataset collected from Kaggle in CSV format.

### 2. Data Preprocessing
- Removed unnecessary columns
- Checked missing values
- Scaled features
- Handled outliers

### 3. Exploratory Data Analysis (EDA)
- Histogram for price distribution
- Scatter plot for area vs price
- Boxplot for outliers
- Heatmap for correlation

### 4. Model Building
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor

### 5. Model Evaluation
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score

---

##  Results
- Random Forest performed best with highest accuracy
- Provided better predictions compared to other models
